HTML & CSS Basics Summary

HTML Summary

HTML(HyperText Markup Language) is used to define my page content, structure and meaning.
You dont't use it for styling purposes, Use CSS for that instead!

  • HTML uses "elements" to describe (annotate) content

    • HTML elements typically have an opening tag, content and then closing tag

  • You can also have void (empty) elements like images

  • You can also configure elements with attributes

    • There's a long list of available elements but you'll gain experience over time, no worries.

Learn more about all available HTML elements on the MDN HTML element reference.

CSS Summary

CSS (Cascading Style Sheets) is used for styling your page content.

    Styles are assigned via property-value pairs

    You can assign styles via the "Style" attribute

  • To avoid code duplication, you typically use globale style (w.g. via the "style" element) instead

    • Alternatively , you can work with external stylesheet files wich ypou "link" to

  • When working with CSS, concepts like "inheritance", "specifity" and the "box model" should be understood.

Learn more about all available CSS properties and values on the MDN CSS property reference.
